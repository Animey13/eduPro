# Predictive Modeling for Course Demand and Revenue Forecasting on EduPro

This project aims to build predictive models for course enrollment demand and revenue forecasting on the EduPro online learning platform.

## Structure
- `data/`: Contains the raw dataset.
- `src/`: Contains source code for data preparation, feature engineering, training, and evaluation.
- `models/`: Contains trained model artifacts.
- `reports/`: Contains the research paper and executive summary.
- `app.py`: Streamlit application for the dashboard.
